# CV基础要点速通

坦白说，有 AI 的今天，OpenCV 基础用法已经没什么好逐条讲的了。你想用摄像头读一帧图像、转成灰度、保存到本地，给 AI 一句话就能出代码，比我在这里写十页都管用。

但有一个东西是 AI 给不了你的——**对于 OpenCV 能做什么、不能做什么的全局认知**。如果你连 OpenCV 有哪些模块、大概能处理什么类型的任务都不知道，你连 prompt 都不知道该怎么写。

所以这一节的目标只有一个：**给你一张 OpenCV 的能力地图，让你以后遇到视觉问题时，至少知道该让 AI 去写哪个方向的代码。**

## 它本质上是一个矩阵运算库

OpenCV 最核心的数据结构是 `cv::Mat`（Python 里是 `numpy.ndarray`）。你在 OpenCV 里做的几乎所有操作——无论是读图、滤波、颜色空间转换、形态学操作、矩阵运算——本质上都是在操作一个多维数组。

理解这一点很重要：当一个视觉问题可以被抽象成"对像素矩阵做数学变换"时，OpenCV 基本都有现成函数。当你需要做的事情超出了"像素矩阵变换"的范畴，OpenCV 就不够用了。

## OpenCV 擅长的

大致可以归类为以下几块，遇到对应的问题可以直接找 AI 要代码：

**图像读写与显示**

- `imread` / `imwrite` / `imshow`——从文件读、写到文件、在窗口显示
- 支持各种格式：jpg、png、bmp 等，也支持从视频文件或摄像头取流

**图像预处理**

- 颜色空间转换：`cvtColor`，BGR↔Gray、BGR↔HSV、BGR↔RGB 等
- 缩放：`resize`，最近邻、双线性、双三次插值等
- 滤波：`blur` / `GaussianBlur` / `medianBlur` / `bilateralFilter`，去噪和平滑
- 形态学操作：`dilate` / `erode` / `morphologyEx`（开闭运算、梯度等）

**特征提取与分析**

- 边缘检测：`Canny` / `Sobel` / `Laplacian`
- 轮廓检测：`findContours` / `drawContours`，配合轮廓筛选
- 直线检测：`HoughLinesP`
- 圆形检测：`HoughCircles`
- 角点检测：`cornerHarris` / `goodFeaturesToTrack`

**几何变换**

- 仿射变换：`warpAffine`，旋转、平移、缩放
- 透视变换：`warpPerspective`，矫正倾斜画面
- 找变换矩阵：`getAffineTransform` / `findHomography`

**目标跟踪与光流**

- CamShift / Meanshift 跟踪
- 光流法：`calcOpticalFlowPyrLK`（稀疏光流）、`Farneback`（稠密光流）

## OpenCV 不擅长的 / 需要其他工具配合的

**语义理解：这是啥东西？**

- OpenCV 知道"这里有一个轮廓"，但它不知道"这是一个人还是一个杯子"
- 这部分属于深度学习范畴，需要 YOLO / 分类网络等模型来解决

**复杂场景下的鲁棒识别**

- 光照剧烈变化、遮挡严重、视角大变——传统 CV 在这些场景下容易崩
- 深度学习模型在鲁棒性上有明显优势，但代价是需要数据和算力

**文本识别（OCR）**

- OpenCV 有基础的 `text` 模块，但效果不如专门的 OCR 引擎
- 实际项目中建议用 Tesseract 或 PaddleOCR 等专用库

**高精度测量**

- OpenCV 的标定和测量功能是有的，但精度受限于相机质量、标定板精度等
- 如果需要工业级精度，还需要专门的计量算法配合

## 所以你的角色是什么

有了 AI 之后，你不再是"写 OpenCV 代码的人"，而是**告诉 AI 需要做什么、检查 AI 做出来的东西是否合理的人**。

具体到 OpenCV 这个层面，你只需要做到：

1. **知道 OpenCV 有哪几类功能**（上面列的那些），这样遇到问题知道往哪个方向问 AI
2. **能读懂 AI 给的代码**，至少知道每行在做什么、数据怎么流的
3. **能调试**——AI 给的阈值、参数经常不是最优的，你得知道调哪里、往哪个方向调

这就够了。剩下的，AI 来写。

## 从哪开始

在本地的 Python 环境里（如果你不知道什么 Python 环境，那就用 uv），执行：

```bash
pip install opencv-python numpy
```

然后就可以跑起来了。随便找一张图片，对着上面列的功能类别，让 AI 给你写 Demo 试试。

```text
我需要将这篇文档里面提到的OpenCV擅长的功能都试一遍，我们一步一步来。先告诉我要实践这些东西，我在场景中需要准备什么。
```

我也没准备过代码，我们直接让AI现场来写并且生成对应的代码。因为这一讲要用摄像头，我把我的皮套人关掉了，下一节再打开。
